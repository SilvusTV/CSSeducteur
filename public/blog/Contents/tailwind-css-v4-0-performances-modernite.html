Nouveau moteur Oxide

Tailwind CSS v4.0 est doté d'un moteur entièrement réécrit, nommé Oxide, conçu pour améliorer considérablement les performances :

  • Performances accrues : Les compilations complètes sont jusqu'à 10 fois plus rapides.
  • Empreinte réduite : Réduction de 35 % de la taille installée.

Chaîne d'outils unifiée

La version 4.0 intègre directement des fonctionnalités auparavant gérées par des plugins externes :

  • Gestion des @import sans postcss-import.
  • Support natif du nesting CSS.
  • Ajout automatique des préfixes vendeurs.

Conçu pour le web moderne

Cette version adopte les fonctionnalités modernes CSS comme :

  • Couches de cascade via @layer.
  • Propriétés CSS personnalisées avec @property.
  • Support des requêtes conteneur via @min-* et @max-*.

Variantes composables

Les nouvelles variantes offrent une expressivité sans précédent :

        
.group-has-focus:opacity-100
.group-not-has-peer-not-data-active:underline

Configuration sans fichier JavaScript

Personnalisez votre thème directement avec CSS :

        
@import "tailwindcss";

@theme {
  --font-family-display: "Satoshi", "sans-serif";
  --color-neon-pink: oklch(71.7% 0.25 360);
  --color-neon-cyan: oklch(91.3% 0.139 195.8);
}
